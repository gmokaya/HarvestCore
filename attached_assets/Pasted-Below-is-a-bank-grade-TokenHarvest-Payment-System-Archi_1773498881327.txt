# 2. Wallet Management System

The **Wallet Engine** is the financial core of TokenHarvest.

Each user automatically receives a **multi-currency wallet**.

### Wallet Structure

```text
User Wallet
│
├ KES Wallet
├ Stablecoin Wallet
└ Escrow Wallet
```

### Functions

The wallet handles:

* deposits
* withdrawals
* loan disbursements
* loan repayments
* trade settlements
* escrow locking

---

### Wallet Database Model

Wallet Table

| Field          | Description       |
| -------------- | ----------------- |
| wallet_id      | Unique wallet ID  |
| user_id        | Owner of wallet   |
| currency       | KES / USDC        |
| balance        | Available balance |
| locked_balance | Funds reserved    |
| created_at     | Creation date     |

Example:

```json
{
 "wallet_id": "WLT-0012",
 "user_id": "FRM-0092",
 "currency": "KES",
 "balance": 120000,
 "locked_balance": 30000
}
```

# 3. Ledger System (Critical)

Every fintech platform requires a **double-entry ledger system**.

This ensures **auditability and reconciliation**.

### Ledger Model

Each transaction creates **two entries**:

```text
Debit Account
Credit Account
```

Example: Loan Disbursement

```text
Liquidity Pool Debit
Borrower Wallet Credit
```

Ledger Table

| Field          | Description         |
| -------------- | ------------------- |
| transaction_id | Unique transaction  |
| debit_account  | Source account      |
| credit_account | Destination account |
| amount         | Transaction amount  |
| currency       | Currency            |
| timestamp      | Date                |

---

# 4. Escrow Engine

The **Escrow Engine** temporarily holds funds during transactions.

Used for:

* commodity trading
* forward contracts
* warehouse receipt purchases

### Escrow Workflow

```text
Buyer deposits funds
        ↓
Funds locked in escrow
        ↓
Commodity delivered
        ↓
Funds released to seller
```

Example:

```text
Buyer Wallet → Escrow Account → Seller Wallet
```
